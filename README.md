# projet-brick-shooter

## Bienvenue sur le jeu Brick Shooter

### Lancer le jeu

#### Windows

##### Créer l'environnement
`py -3 -m venv venv`
##### Activer l'environnement
`venv\Scripts\activate`
##### Lancer le serveur
`py -3 app.py`
<br>

#### Mac OS

##### Créer l'environnement
`python3 -m venv venv`
##### Activer l'environnement
`. venv/bin/activate`
##### Lancer le serveur
`python3 app.py`

### Règles du jeu

Le but du jeu est de détruire tous les blocs sans qu'ils ne vous touchent !
Pour cela il suffit de vous déplacez de gauche à droite avec les flèches de votre 
clavier et de tirer avec espace.
Tirer sur les blocs pour gagner des crédits et utilisez-les pour améliorer vos données dans Equipement.

Les blocs :

Orange : augmente la vitesse de défilement <br>
Vert : diminue la vitesse de déplacement <br>
Violet : diminue la vitesse de tir <br>
Blanc: neutre <br>
